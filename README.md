# Library-Management-System
A robust, console-based Library Management System developed in C++. This project focuses on managing a library database with two levels of access (Admin and User), featuring real-time data persistence using file handling. It demonstrates efficient use of dynamic memory allocation and structured logic to handle book records.
### Key Features:
* **Dual Access:** Admin (Password protected) and User modes.
* **Full CRUD:** Add, update, remove, and display books.
* **Borrowing System:** Real-time tracking of book availability.
* **Data Persistence:** Saves all changes to `library_data.txt`.

### Technical Skills:
* **Procedural Programming** & Modular Functions.
* **File I/O** for permanent storage.
* **Dynamic Memory Management** (new/delete).
